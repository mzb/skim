              Syntax restrictions
              ===================
                - Variable declaration WITHOUT INITIALIZATION
                    var? NAME : int | float | bool | string | [size]*
                - Variable assignment
                    NAME = PRIMITIVE-LITERAL | [] | FUNCTION-CALL
                - Array assignment
                    array[idx] = PRIMITVE-LITERAL | [] | FUNCTION-CALL
                - Multidimensional arrays
                    var a:int[2][3] -> vector<vector<int>> a(2);
                    a[0] = []       -> a[0] = vector<int>()
                    a[1][1] = 5     -> a[1][1]
                - NO ARRAY LITERALS EXCEPT []
                - SEMICOLONS
